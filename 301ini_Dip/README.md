启动信道探测测距功能的完整流程：

初始化蓝牙子系统。
启动蓝牙扫描，发现支持 Ranging Service 的设备。
建立连接并完成服务发现。
配置信道探测的默认参数和详细参数。
启用信道探测的安全性。
设置测距参数并启动测距过程。
在主循环中处理测距数据。

需在ranging_data_get_complete_cb下对cs_de_populate_report重构，以自定义数据存储的方式。

## 条件编译说明

本项目在 `main.c` 中使用了多个条件编译宏来控制不同的功能模式。以下是各个条件编译选项的详细说明：

### 1. FLASH_WRITE_MODE (Flash写入模式)

**定义位置**: `src/flash/flash_ops.h` 第30行

**可选值**:
- `FLASH_WRITE_MODE_SINGLE` (0): 单个写入模式
- `FLASH_WRITE_MODE_BATCH` (1): 批量写入模式

**当前默认值**: `FLASH_WRITE_MODE_SINGLE`

**作用**:
- **单个写入模式 (SINGLE)**: 每次测距完成后，立即使用 `k_work` 异步写入单个数据记录到 Flash。适合对实时性要求较高的场景，但写入频率较高可能影响性能。
- **批量写入模式 (BATCH)**: 使用环形缓冲区缓存多个测距数据，当缓冲区达到阈值（27个记录，约1.35秒数据）或缓冲区满时，批量写入 Flash。同时启动定时刷新机制（每10秒检查一次），确保数据及时持久化。适合需要提高写入效率、减少 Flash 磨损的场景。

**影响范围**:
- 数据写入方式的选择（`ranging_data_get_complete_cb` 函数）
- 是否定义定时刷新缓冲区的工作队列
- 是否初始化并启动定时刷新机制

### 2. TEST_RANGING_ENABLED (测试测距功能)

**定义位置**: `src/main.c` 第83行

**可选值**:
- `0`: 禁用测试测距功能（默认）
- `1`: 启用测试测距功能

**作用**:
启用后，系统会进行固定次数的测距测试（默认25次，可通过 `TEST_RANGING_COUNT` 修改），并统计以下信息：
- 总尝试次数 (`ranging_count`)
- 成功获取数据的次数 (`ranging_success_count`)
- 数据获取失败的次数 (`ranging_error_count`)
- 数据被覆盖的次数 (`ranging_overwritten_count`)

测试完成后，系统会自动禁用信道探测功能，并输出成功率、错误率、覆盖率等统计信息。

**影响范围**:
- 定义测试相关的计数器变量
- 在测距回调中记录和统计测试数据
- 在数据覆盖回调中记录覆盖次数
- 在按键处理中重置测试计数器
- 在主循环中显示测试进度

### 3. ENABLE_DIRECT_PRINT (直接打印模式)

**定义位置**: `src/flash/flash_ops.h` 第33行

**可选值**:
- `0`: 禁用直接打印模式，数据写入 Flash（默认）
- `1`: 启用直接打印模式，测距完成后直接打印到串口，不写入 Flash

**作用**:
启用后，测距数据获取完成后会直接通过串口打印输出，而不保存到 Flash。适用于调试场景，可以实时查看测距结果，但数据不会被持久化存储。

**影响范围**:
- `ranging_data_get_complete_cb` 函数中的数据存储逻辑

### 4. NEED_LOG (日志输出控制)

**定义位置**: `src/main.c` 第92行

**可选值**:
- `0`: 禁用详细日志输出（默认，用于提升系统性能）
- `1`: 启用详细日志输出

**作用**:
控制主循环中是否定期输出距离估计信息。启用后，系统会每秒输出各天线路径的距离估计值（ifft、phase_slope、rtt）。禁用可以减少日志开销，提升系统实时性能。

**影响范围**:
- 主循环中的距离估计日志输出

### 使用建议

1. **生产环境**: 建议使用 `FLASH_WRITE_MODE_BATCH`（批量写入模式），`TEST_RANGING_ENABLED=0`，`ENABLE_DIRECT_PRINT=0`，`NEED_LOG=0`，以获得最佳性能和稳定性。

2. **调试环境**: 可以启用 `ENABLE_DIRECT_PRINT=1` 和 `NEED_LOG=1`，方便实时查看测距结果和系统状态。

3. **性能测试**: 启用 `TEST_RANGING_ENABLED=1`，可以自动进行固定次数的测距测试并获取统计信息。
