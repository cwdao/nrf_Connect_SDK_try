主要功能
SPI Flash 操作：

使用 Zephyr 的 flash 驱动对 SPI Flash 存储器进行读写和擦除操作。
支持日志记录功能，日志数据通过特定的结构体存储到 Flash 中。
按钮控制：

程序通过 4 个按钮控制不同的功能：
Button 0：启动日志写入功能。
Button 1：停止日志写入/读取功能。
Button 2：启动日志读取功能。
Button 3：擦除 Flash 存储器。
LED 指示：

使用两个 LED 指示操作状态：
LED 0：在执行 Flash 操作时点亮，操作完成后熄灭。
LED 1：未明确使用，但初始化为输出低电平。
日志记录结构：

日志数据以结构体形式存储，包括测量值、距离估计、质量指标等。数据由 cs_de_report_t 定义。
